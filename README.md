## Run Locally

Clone the project

```bash
  git clone https://github.com/Meta-Dex-Platform/E-commerce
```

Go to the project directory

```bash
  cd React_E-Commerce
```

Install dependencies

```bash
  npm install --legacy-peer-deps

  OR 

  npm install react-material-ui-carousel --save --legacy-peer-deps
```
Run the project

```bash
  npm start
```
